Harmony Ready

## Typical Usage  
  
Clone this repository, develop the contract + tests.  
  
```
yarn
yarn compile
yarn test
```
  
Note:  
The .env data is not sensitive in this project. However, do NOT upload your own if you deploy using a real wallet :)
